Esta é a entrega de número 3 para a disciplina de Desenvolvimento Front-End para WEB do Curso de Ciências da Computação pela Universidade Cruzeiro do Sul.

É referente à implementação de estrutura em JavaScript para o HTML da ONG fictícia "Sonora Missão Musical", responsável pelo ensino de música à crianças em situação de vulnerabilidade, assim como, acompanhamento psicossocial.
